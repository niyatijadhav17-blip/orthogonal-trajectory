How Orthogonal Trajectories Work

Orthogonal trajectories are curves that intersect a given family of curves at right angles. To find them:

1. Start with a family of curves F(x, y, a) = 0, where 'a' is a parameter.

2. Differentiate to eliminate 'a' and obtain a differential equation.

3. Replace dy/dx with -dx/dy to get the differential equation for orthogonal trajectories.

4. Solve this new differential equation to find the orthogonal family.

In this tool, we use numerical methods to calculate and visualize these trajectories for the given curve family.
