How Orthogonal Trajectories Work

Orthogonal trajectories are curves that intersect a given family of curves at right angles (90°).

Mathematical Procedure:

1. Start with a family of curves F(x, y, a) = 0, where 'a' is a parameter.

2. Differentiate implicitly with respect to x to eliminate 'a' and obtain a differential equation: dy/dx = f(x, y).

3. For orthogonal trajectories, replace dy/dx with -dx/dy (or -1/(dy/dx)).

4. Solve this new differential equation to find the orthogonal family.

This tool uses both analytical solutions (for standard curves) and numerical methods (for complex equations) to calculate orthogonal trajectories.
